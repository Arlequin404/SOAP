# SOAP Hello World
Este proyecto es una demostración básica de un servicio SOAP en PHP.

## Archivos importantes
- `soap-server.php`: Servidor SOAP que expone una función `sayHello`.
- `test-soap.php`: Cliente que consume el servicio SOAP.
- `soap.wsdl`: Archivo WSDL que describe el servicio.
- `index.php`: Página principal para navegación.
